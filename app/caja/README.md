# Caja Web

Este directorio conserva el modulo actual de Caja basado en React + Vite.

## Flujo real

- Frontend activo: `app/caja/web`
- API y Socket.IO: `services/backend/server.js`
- Puerto local del frontend: `5175`

## Comandos

```bash
npm run dev
npm run build
npm run preview
```

Los comandos de este directorio son wrappers hacia `app/caja/web` para mantener una entrada simple.
